# 📚 Smart Library Management System

## 📌 Overview
The Smart Library Management System is a Java-based application with a graphical user interface (GUI) that helps manage library operations efficiently. It uses MySQL for data storage and includes a secure login system for authentication.

## 🎯 Objective
To automate library tasks like managing books, users, and transactions while reducing manual effort.

## 🚀 Features
- 🔐 User Login Authentication
- 📝 User Registration System
- 📖 View Books
- ➕ Add Books
- ❌ Delete Books
- 🔄 Issue and Return Books
- 🖥️ GUI Interface using Java Swing
- 🗄️ MySQL Database Integration
- 👨‍💻role-based access (Admin/User)
## 🛠️ Technologies Used
- Java (Core Java + Swing)
- MySQL
- JDBC (Java Database Connectivity)

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Event Handling (for GUI)
- Database Connectivity (JDBC)
- Authentication System

## 📂 Project Structure
- Main.java → Entry point
- Login.java → Handles login authentication
- Register.java → Handles new user registration
- Dashboard.java → Main UI after login
- Book.java → Book model
- DBConnection.java → Database connection

## ▶️ How to Run
1. Install Java (JDK)
2. Install MySQL and create database
3. Update DB username & password in `DBConnection.java`
4. Open project in Eclipse/IntelliJ
5. Run `Main.java`

## 🗄️ Database Setup
- Create a database (e.g., `library_db`)
- Create tables for:
  - users
  - books
